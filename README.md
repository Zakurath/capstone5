# AI Threat Intelligence Aggregator

## Project Overview
This project aggregates AI-related threat intelligence from multiple sources
(MITRE ATLAS, CISA KEV, academic research) and categorizes them into:
- Hypothetical
- Demonstrated
- Active Exploitation

The goal is to provide a centralized feed to help security teams prioritize AI risks.

## Data Sources
- CISA Known Exploited Vulnerabilities (KEV)
- MITRE ATLAS
- NIST NVD (CVE enrichment)

## Project Structure
/docs        -> documentation
/data        -> raw and processed threat data
/src         -> ingestion, processing, API logic, gui
